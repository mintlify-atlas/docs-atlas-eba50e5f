## Overview

Benefits are perks granted to customers who purchase products. They can be license keys, downloadable files, Discord roles, GitHub repository access, and more.

## The Benefit Object

<ResponseField name="id" type="string" required>
  Unique identifier for the benefit
</ResponseField>

<ResponseField name="type" type="enum" required>
  Type: `custom`, `discord`, `github_repository`, `downloadables`, `license_keys`, `ads`, or `feature_flag`
</ResponseField>

<ResponseField name="description" type="string" required>
  Benefit description shown to customers
</ResponseField>

<ResponseField name="organization_id" type="string" required>
  Organization ID
</ResponseField>

<ResponseField name="properties" type="object" required>
  Type-specific configuration
</ResponseField>

<ResponseField name="is_tax_applicable" type="boolean" required>
  Whether tax applies to this benefit
</ResponseField>

<ResponseField name="metadata" type="object">
  Custom metadata
</ResponseField>

<ResponseField name="created_at" type="string" required>
  Creation timestamp
</ResponseField>

## Benefit Types

### Custom Benefit

A generic benefit you manage manually:

```json
{
  "type": "custom",
  "description": "Priority support",
  "properties": {
    "note": "Grant access in your system"
  }
}
```

### License Keys

Automatically generated license keys:

```json
{
  "type": "license_keys",
  "description": "Software license",
  "properties": {
    "prefix": "PROD",
    "expires": 31536000,
    "activations": 3,
    "limit_usage": true
  }
}
```

### Downloadables

Downloadable files:

```json
{
  "type": "downloadables",
  "description": "Premium templates",
  "properties": {
    "files": ["file_123", "file_456"]
  }
}
```

### Discord Role

Automatic Discord role assignment:

```json
{
  "type": "discord",
  "description": "VIP Discord role",
  "properties": {
    "guild_id": "123456789",
    "role_id": "987654321"
  }
}
```

### GitHub Repository Access

Grant access to private repositories:

```json
{
  "type": "github_repository",
  "description": "Access to private repo",
  "properties": {
    "repository_owner": "myorg",
    "repository_name": "private-repo",
    "permission": "pull"
  }
}
```
